# Tienda Muebles Backend

Backend API REST en .NET Core 10. Proyecto sincronizado con el vault de Obsidian.

## Stack

- .NET 10
- Entity Framework Core  
- Clean Architecture
- SQL Server

## Estructura

src/           - Codigo fuente
tests/         - Tests unitarios e integracion
docs/          - Documentacion (sincronizada con vault)
  specs/       - Especificaciones
  updates/     - Reportes de progreso
  decisiones/  - ADRs

## Sincronizacion
